### 내장 스킬 목록 보기

채팅창에 아래 프롬프트를 입력합니다:

```
'/'을 쳤을 때 상단에 '기술'탭이 보임
```

또는:

```
내 스킬 목록을 보여줘
```

### 확인 가능한 정보

* **이름(name):** 스킬 식별자 (영문 kebab-case)
* **설명(description):** 어떤 상황에서 이 스킬이 발동되는지



### 내 OneDrive의 개인 스킬 위치

개인 스킬 파일은 OneDrive의 다음 경로에 저장됩니다:

```
Documents/Cowork/
```

> 파일 관리자(OneDrive 웹 또는 파일 탐색기)로 이 폴더에 접근해 기존 스킬을 확인하고 추가할 수 있습니다.
