### 5-1. 트리거 문장으로 호출

등록한 트리거 문장을 그대로 채팅창에 입력합니다:

```
EV 자동차 시장에 대한 최근 한달 내의 최신 뉴스를 정리해서 HTML 대시보드를 만들어서 나에게 메일로 전송해. 
메일 내용은 본부장님에게 보낼 수 있도록 작성해줘. 

그리고 EV 자동차 관련 한달 내의 최신 뉴스와 함께 우리 내부 EV 충전소 확장을 위한 프로젝트 자료인 /EV 충전소 데이터_2025.xlsx 와 EV 충전소 인사이트.docx
문서를 참고하여 팀장님에게 보고할 PPT 문서를 만들어줘. 
```

### 5-2. 슬래시 명령어로 직접 호출

스킬 이름 앞에 `/`를 붙여 직접 실행합니다:

```
/exec-report-ppt 을 사용하여 최신 전기 자동차 산업 조사하고 보고서 만들어줘.
```

### 5-3. 파라미터와 함께 사용

스킬에 추가 정보를 함께 전달할 수 있습니다:

```
최근 한달 내의 국내 ESG 현황을 담은 보고서 작성해줘. 특히 EV 자동차 충전소 품질보증 관련 업무 중심으로 정리해줘.
```

### 5-4. 스킬 수정

스킬 동작을 개선하고 싶을 때:

```
exec-report-ppt 스킬을 수정해줘.
맨 마지막 슬라이드에 end of document를 추가해줘
```

### 5-5. 스킬 삭제

```
exec-report-ppt 스킬을 삭제해줘
```
