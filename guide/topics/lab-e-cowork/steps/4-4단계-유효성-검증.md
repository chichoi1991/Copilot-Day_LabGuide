스킬을 등록한 후 아래 프롬프트로 검증합니다.

### 4-1. 스킬 목록에서 확인

```
내 스킬 목록을 보여줘
```

새로 등록한 스킬이 목록에 나타나는지 확인합니다.

### 4-2. 스킬 상세 검증

```
 스킬을 검증해줘
```

또는:

```
내 스킬 파일에 문제가 있는지 확인해줘
```

Copilot이 확인하는 항목:

|검사 항목|정상 상태|
|-|-|
|YAML 프론트매터 형식|`---` 블록이 올바르게 닫혀 있음|
|`name` 필드|영문 소문자, 하이픈만 사용|
|`description` 필드|한 줄, 명확한 설명|
|`triggers` 목록|2개 이상의 트리거 문장|
|스킬 본문|실행 지침이 충분히 구체적임|
|트리거 충돌|다른 스킬과 트리거가 겹치지 않음|
